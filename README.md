# DCGAN From Scratch

A PyTorch implementation of a **Deep Convolutional Generative Adversarial Network (DCGAN)** built entirely from scratch for generating handwritten digits from the **MNIST** dataset. This project demonstrates the complete adversarial training pipeline by implementing both the Generator and Discriminator architectures using convolutional neural networks.

---

## Overview

Generative Adversarial Networks (GANs) consist of two neural networks trained in a competitive setting:

* **Generator** learns to synthesize realistic handwritten digit images from random latent vectors.
* **Discriminator** learns to distinguish between real MNIST images and images generated by the Generator.

Through adversarial training, the Generator progressively learns the underlying distribution of the dataset and produces increasingly realistic samples.

## Model Architecture

### Generator

The Generator transforms a **100-dimensional latent vector** into a grayscale MNIST digit using learned upsampling layers.

**Architecture**

```
Latent Vector (100)
        │
Fully Connected Layer
        │
Reshape → 128 × 4 × 4
        │
ConvTranspose2D
        │
BatchNorm + ReLU
        │
ConvTranspose2D
        │
BatchNorm + ReLU
        │
ConvTranspose2D
        │
Sigmoid
        │
Generated Image
```

---

### Discriminator

The Discriminator is a convolutional neural network that classifies an input image as **Real** or **Fake**.

**Architecture**

```
Input Image
      │
Conv2D
      │
BatchNorm + ReLU
      │
Conv2D
      │
BatchNorm + ReLU
      │
Conv2D
      │
BatchNorm + ReLU
      │
Conv2D
      │
Flatten
      │
Linear Layer
      │
Real / Fake Score
```

## Training Workflow

1. Sample a random latent vector from a Gaussian distribution.
2. Generate synthetic digit images using the Generator.
3. Train the Discriminator on both real and generated images.
4. Compute adversarial loss.
5. Update the Generator to fool the Discriminator.
6. Repeat until convergence.
